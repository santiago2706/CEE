# CEE Web

Plataforma web institucional para el Centro de Especializacion Ejecutiva (CEE-FIIS).

## Stack

- Turborepo + pnpm workspaces
- React + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- Zustand, Axios y React Router

## Apps

- `apps/web`: sitio publico CEE.
- `apps/admin`: panel backoffice CEE.

## Paquetes

- `packages/types`: tipos TypeScript compartidos.
- `packages/config`: configuracion TypeScript compartida.
- `packages/ui`: componentes compartidos.

## Comandos

```bash
pnpm install
pnpm dev
pnpm build
```
